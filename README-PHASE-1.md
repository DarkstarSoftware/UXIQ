# Darkstar Audit AI — Phase 1 Landing Funnel

This package updates the public landing experience.

## Goal

Make the landing page informative, conversion-focused, and clear that users must create an account or sign in before analyzing a website.

## Included

- Updated landing page
- Updated pricing page
- Updated metadata
- Updated logo component

## Apply

Copy these files into the root of your current project and replace existing files.

Then run:

```bash
npm run build
git add .
git commit -m "Phase 1 landing page conversion funnel"
git push
```
